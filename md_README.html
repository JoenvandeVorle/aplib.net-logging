Aplib.Net

Overview

Aplib.Net is a C# re-implementation of the Agent Programming LIBrary (Aplib) for Java. The Java implementation of Aplib can be found here and is licensed to Utrecht University, created under the iv4XR project.

Aplib.Net provides a set of tools for creating automatic playtesting agents for games. It is designed in such a way that it can be used in any game that uses the C# language. Additionally, we provide support for the Unity game engine, due to its large user base.

A demo game, showcasing the functionality of the library, can be found in this repository.

Installation

Aplib.Net is available as a NuGet package or as a simple DLL file, that can then be added as a reference.

Both are available in the GitHub Release.

An easier way of installing is also provided using the NuGet Gallery at NuGet.org, which can be found here.

Unity Installation

Unity installation is a bit tricky, but can be done in two ways. The first, and easier way is to install the NuGet plugin and import it via NuGet.

For a manual installation, a few steps must be taken. As Unity does not support external libraries in .NET Standard API, the API Compatibility level must be set to .NET Framework in Project Settings > Player > Configuration > Api Compatibility Level.

Download the latest DLL from the GitHub Release and place it in Assets/Plugins. In the root of your project, add a new file called csc.rsp and add the following line:

-r:Assets/Plugins/Aplib.Core.dll
